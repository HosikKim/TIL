# 알고리즘 선택 가이드

> 코딩테스트 문제를 풀 때 빠르게 참고할 수 있는 알고리즘 선택 가이드

## 조건별 알고리즘 선택표

| 조건 (이것 보이면) | 1순위 | 대안 | 복잡도 |
|-------------------|-------|------|--------|
| **N ≤ 10** (모든 경우 확인) | 완전탐색 (N!) | 백트래킹 | O(N!) |
| **N ≤ 20** (부분집합/조합) | 비트마스킹 (2^N) | 재귀 | O(2^N) |
| "최단거리", "최소이동" (가중치 없음) | BFS | 다익스트라 | O(V+E) |
| "모든 경로", "경로 존재 여부", 연결 요소 | DFS | Union-Find | O(V+E) |
| **가중치 있는** 최단거리 (E log V 가능) | 다익스트라 | 벨만포드 | O(E log V) |
| "최대/최소" (이전 결과 재사용) | DP | 그리디 (증명 가능 시) | 문제 의존 |
| 정렬된 데이터 + "~를 만족하는 최소/최대" | 이분탐색 | 투포인터 | O(log N) |
| 연속 구간 합/곱, 슬라이딩 | 투포인터 | 누적합 | O(N) |
| "사이클 검출", "위상 정렬" | DFS / 위상정렬 | - | O(V+E) |
| "구간 쿼리" (업데이트 빈번) | 세그먼트 트리 | 펜윅 트리 | O(log N) |

## 유형별 핵심 정리

| 유형 | 핵심 개념 | 대표 키워드 | 대표 문제 예시 |
|------|----------|------------|--------------|
| **정렬 / 이분탐색** | 기본 정렬 활용 | "정렬된", "~번째", "범위" | 수 찾기, 공유기 설치 |
| **스택 / 큐** | LIFO, FIFO 구조 활용 | "최근", "순서대로", "대기" | 괄호 검사, 프린터 큐 |
| **DFS / BFS** | 그래프/트리 탐색, 방문 처리 | "경로", "연결", "탐색" | 미로 탐색, 연결 요소 |
| **투포인터 / 슬라이딩 윈도우** | 구간 합, 연속 부분 문자열 | "연속", "부분", "구간" | 두 수의 합, 부분 문자열 |
| **최단경로** | 다익스트라, Union-Find | "최단", "최소 비용", "거리" | 그래프 최단거리, 네트워크 |

## 빠른 판단 체크리스트

### 1. 먼저 데이터 크기 확인
```
N ≤ 10        → 완전탐색, 백트래킹
N ≤ 20        → 비트마스킹, 재귀
N ≤ 100       → O(N³) DP, 플로이드-워셜
N ≤ 1,000     → O(N²) DP, 그래프 탐색
N ≤ 10,000    → O(N² log N), 정렬 기반
N ≤ 100,000   → O(N log N), 이분탐색, 투포인터
N ≤ 1,000,000 → O(N), O(N log N), 해시, 정렬
```

### 2-1 키워드로 유형 판단

| 키워드 그룹 | → | 알고리즘 |
|-----------|---|---------|
| "최단", "최소 이동", "가장 빠른" | → | **BFS** (가중치 X) / **다익스트라** (가중치 O) |
| "모든 경로", "존재 여부", "연결" | → | **DFS** |
| "최대/최소", "가장 큰/작은", "최적" | → | **DP** / **그리디** |
| "정렬된", "~번째", "범위" | → | **이분탐색** |
| "연속", "부분 배열", "구간" | → | **투포인터** / **슬라이딩 윈도우** |
| "괄호", "최근", "LIFO" | → | **스택** |
| "순서대로", "대기", "FIFO" | → | **큐** |

### 2-2 그래프 문제 판단

| 문제 유형 | 그래프 종류 | 추천 알고리즘 |
|----------|-----------|--------------|
| 가중치 없는 최단 거리 | 무방향/방향 그래프 | BFS |
| 가중치 있는 최단 거리 | 양수 가중치 | 다익스트라 |
| 음수 가중치 포함 | 음수 가중치 | 벨만포드 |
| 모든 정점 쌍 최단 거리 | Dense 그래프 | 플로이드-워셜 |
| 연결 요소 개수 | 무방향 그래프 | DFS / Union-Find |
| 사이클 검출 | 방향/무방향 그래프 | DFS |
| 위상 정렬 | DAG | 위상 정렬 (Kahn) |



### 3. 시간복잡도 우선순위
```
O(1) > O(log N) > O(N) > O(N log N) > O(N²) > O(N³) > O(2^N) > O(N!)
```
