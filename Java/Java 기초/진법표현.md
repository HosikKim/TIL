# Java 진법 표현

## 개념

정수를 다양한 진법으로 출력하는 형식 지정자

## 기본 문법
```java
int num = 15;

System.out.printf("%d \n", num);  // 15 (10진수)
System.out.printf("%o \n", num);  // 17 (8진수)
System.out.printf("%x \n", num);  // f (16진수)
System.out.printf("%X \n", num);  // F (16진수 대문자)
```

## 형식 지정자

| 지정자 | 진법 | 표현 범위 | 예시 (15) |
|--------|------|----------|-----------|
| %d | 10진수 | 0-9 | 15 |
| %o | 8진수 | 0-7 | 17 |
| %x | 16진수 소문자 | 0-9, a-f | f |
| %X | 16진수 대문자 | 0-9, A-F | F |

## 접두어 추가
```java
int num = 255;

System.out.printf("%#o \n", num);   // 0377 (8진수)
System.out.printf("%#x \n", num);   // 0xff (16진수)
System.out.printf("%#X \n", num);   // 0XFF (16진수 대문자)
```

## 변환 예시

| 10진수 | 8진수 (%o) | 16진수 (%x) |
|--------|-----------|------------|
| 8 | 10 | 8 |
| 15 | 17 | f |
| 16 | 20 | 10 |
| 255 | 377 | ff |

## 주요 사용 사례

### 8진수 (%o)

**파일 권한 (Unix/Linux)**:
```java
int permission = 0644;
System.out.printf("권한: %o\n", permission);  // 644
// rw-r--r-- 의미
```

### 16진수 (%x, %X)

**1. 메모리 주소**:
```java
Object obj = new Object();
System.out.printf("주소: %x\n", 
    System.identityHashCode(obj));
```

**2. 색상 코드**:
```java
int red = 255, green = 87, blue = 51;
System.out.printf("#%02X%02X%02X\n", red, green, blue);
// #FF5733
```

**3. MAC 주소**:
```java
byte[] mac = {0x00, 0x1A, 0x2B, 0x3C, 0x4D, 0x5E};
for (byte b : mac) {
    System.out.printf("%02X:", b);
}
// 00:1A:2B:3C:4D:5E
```

**4. 바이트 데이터 디버깅**:
```java
byte[] data = {0x48, 0x65, 0x6C, 0x6C, 0x6F};
for (byte b : data) {
    System.out.printf("%02X ", b);
}
// 48 65 6C 6C 6F
```

## 실용 예시

### 비트 연산 결과 확인
```java
int a = 0b1010;  // 2진수
int b = 0b1100;

System.out.printf("AND: %x\n", a & b);  // 8
System.out.printf("OR: %x\n", a | b);   // e
System.out.printf("XOR: %x\n", a ^ b);  // 6
```

### 데이터 변환 확인
```java
String hex = "FF";
int decimal = Integer.parseInt(hex, 16);
System.out.printf("16진수 %s = 10진수 %d\n", hex, decimal);
// 16진수 FF = 10진수 255
```

## 정리

**사용 이유**:
- 10진수: 일반적인 숫자 표현
- 8진수: 파일 권한, 비트 패턴
- 16진수: 메모리, 색상, 네트워크, 디버깅

**선택 기준**:
- 일반 출력 → %d
- 파일 권한 → %o
- 메모리/색상/바이트 → %x
