# 제네릭 (Generic)

## 개념

타입을 파라미터로 사용하여 타입 안정성과 코드 재사용성을 높이는 기능

**목적**: 컴파일 시점 타입 체크, 타입 캐스팅 제거

## 제네릭 사용 전후

### 사용 전
```java
List list = new ArrayList();
list.add("문자열");
String str = (String) list.get(0);  // 캐스팅 필요
list.add(123);  // 런타임 에러 위험
```

### 사용 후
```java
List<String> list = new ArrayList<>();
list.add("문자열");
String str = list.get(0);  // 캐스팅 불필요
// list.add(123);  // 컴파일 에러
```

## 제네릭 클래스
```java
public class Box<T> {
    private T item;
    
    public void set(T item) { this.item = item; }
    public T get() { return item; }
}

// 사용
Box<String> box = new Box<>();
box.set("Hello");
String str = box.get();
```

## 타입 파라미터 명명

| 기호 | 의미 |
|------|------|
| T | Type |
| E | Element |
| K | Key |
| V | Value |

## 다중 타입 파라미터
```java
public class Pair<K, V> {
    private K key;
    private V value;
    
    public Pair(K key, V value) {
        this.key = key;
        this.value = value;
    }
}

// 사용
Pair<String, Integer> pair = new Pair<>("나이", 25);
```

## 제네릭 메서드
```java
public static <T> void printArray(T[] array) {
    for (T element : array) {
        System.out.print(element + " ");
    }
}

// 사용
String[] strArray = {"A", "B", "C"};
printArray(strArray);  // A B C
```

## 제한된 타입 파라미터

### extends (상한 제한)
```java
// Number와 하위 클래스만 허용
public <T extends Number> double sum(List<T> list) {
    double total = 0.0;
    for (T num : list) {
        total += num.doubleValue();
    }
    return total;
}
```

### 와일드카드 (?)
```java
// 모든 타입
public void printList(List<?> list) {
    for (Object obj : list) {
        System.out.println(obj);
    }
}

// 상한 제한
List<? extends Number> list;  // Number와 하위

// 하한 제한
List<? super Integer> list;  // Integer와 상위
```

## 장점

| 장점 | 설명 |
|------|------|
| 타입 안정성 | 컴파일 시점 타입 체크 |
| 캐스팅 제거 | 자동 타입 변환 |
| 재사용성 | 여러 타입 처리 |
| 가독성 | 의도 명확 |

## 주의사항

### 1. static 멤버 사용 불가
```java
public class Box<T> {
    // static T item;  // ❌
}
```

### 2. 배열 생성 불가
```java
// T[] array = new T[10];  // ❌
List<T> list = new ArrayList<>();  // ✅
```

### 3. 타입 소거
- 컴파일 후 제네릭 타입 정보 제거
- 런타임에는 타입 알 수 없음
