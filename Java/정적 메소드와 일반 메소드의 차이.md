# 정적 메서드 vs 일반 메서드

## 비교표

| 구분 | 정적 (static) | 일반 (instance) |
|------|--------------|-----------------|
| 선언 | static 키워드 | static 없음 |
| 호출 | 클래스명.메서드() | 객체.메서드() |
| 메모리 | 클래스 로딩 시 1회 | 객체마다 할당 |
| 접근 | static 멤버만 | 모든 멤버 |
| this | 사용 불가 | 사용 가능 |
| 오버라이딩 | 불가 | 가능 |
| 속도 | 빠름 | 상대적 느림 |

## 호출 방식

### 정적 메서드
```java
class Math {
    static int add(int a, int b) {
        return a + b;
    }
}

// 객체 생성 없이 호출
int result = Math.add(5, 3);
```

### 일반 메서드
```java
class Calculator {
    int add(int a, int b) {
        return a + b;
    }
}

// 객체 생성 후 호출
Calculator calc = new Calculator();
int result = calc.add(5, 3);
```

## 접근 가능한 멤버

### 정적 메서드
```java
class MyClass {
    static int staticVar = 10;
    int instanceVar = 20;
    
    static void staticMethod() {
        System.out.println(staticVar);      // ✅ 가능
        System.out.println(instanceVar);    // ❌ 불가
        instanceMethod();                   // ❌ 불가
    }
    
    void instanceMethod() { }
}
```

### 일반 메서드
```java
class MyClass {
    static int staticVar = 10;
    int instanceVar = 20;
    
    void instanceMethod() {
        System.out.println(staticVar);      // ✅ 가능
        System.out.println(instanceVar);    // ✅ 가능
        staticMethod();                     // ✅ 가능
    }
    
    static void staticMethod() { }
}
```

## this 키워드
```java
class MyClass {
    int value = 10;
    
    static void staticMethod() {
        // System.out.println(this.value);  // ❌ this 사용 불가
    }
    
    void instanceMethod() {
        System.out.println(this.value);     // ✅ this 사용 가능
    }
}
```

## 오버라이딩
```java
class Parent {
    static void staticMethod() {
        System.out.println("Parent static");
    }
    
    void instanceMethod() {
        System.out.println("Parent instance");
    }
}

class Child extends Parent {
    // 오버라이딩 아님 (hiding)
    static void staticMethod() {
        System.out.println("Child static");
    }
    
    // 오버라이딩 ✅
    @Override
    void instanceMethod() {
        System.out.println("Child instance");
    }
}
```

## 용도

### 정적 메서드
- 유틸리티 함수
- 팩토리 메서드
- 객체 상태와 무관한 기능
```java
// 유틸리티
Math.max(10, 20);
Integer.parseInt("123");

// 팩토리
LocalDate.now();
Arrays.asList(1, 2, 3);
```

### 일반 메서드
- 객체 상태 변경
- 인스턴스 변수 사용
- 객체 고유 동작
```java
List<String> list = new ArrayList<>();
list.add("item");        // 객체 상태 변경
list.size();             // 객체 상태 조회
```

## 선택 기준

**정적 메서드 사용**:
- 객체 상태 불필요
- 유틸리티 성격
- 전역 접근 필요

**일반 메서드 사용**:
- 객체 상태 필요
- 다형성 활용
- 인스턴스별 동작
